# Desafio 100 Days of Code

## Objetivo

Desenvolver consistência, disciplina e capacidade de resolução de problemas através da programação diária durante 100 dias consecutivos.

A ideia surgiu após completar 230 dias seguidos no Duolingo. Isso me fez refletir sobre o impacto que a mesma disciplina poderia gerar na minha evolução como desenvolvedor de software.

Mais do que aprender linguagens ou acumular commits, este desafio tem como foco criar o hábito de programar todos os dias, independentemente das circunstâncias.

---

## Regras do Desafio

* Resolver pelo menos um problema ou desenvolver algo relacionado à programação todos os dias durante 100 dias consecutivos.
* Utilizar principalmente as linguagens C e Python.
* Praticar através de plataformas como LeetCode, Beecrowd e projetos próprios.
* Não utilizar Inteligência Artificial para gerar soluções prontas.
* A IA pode ser utilizada apenas para explicação de conceitos, lógica, documentação e aprendizado.
* Registrar o progresso diariamente.

---

## Motivação

Estou em transição de carreira para a área de tecnologia e buscava um desafio que colocasse à prova minha disciplina e comprometimento com o aprendizado.

O objetivo não é atingir a perfeição nem manter uma sequência para impressionar outras pessoas. O objetivo é evoluir um pouco a cada dia.

Enquanto muitas pessoas focam apenas no resultado final, quero focar no processo.
